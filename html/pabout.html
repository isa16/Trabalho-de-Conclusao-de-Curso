Tipos de Choro

Tipo 1

Quando o bebê chora emitindo o som "nhê", é o reflexo que o bebê faz para sugar.

Tipo 2

Quando o bebê chora emitindo o som "aa", é reflexo que o bebê tem para bocejo.

Tipo 3

Quando o bebê emite o som "heh", é o reflexo de desconforto.

Tipo 4

Quando o bebê emite o som "éar", é para emitir dor.
